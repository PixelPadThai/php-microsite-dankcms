# AGENTS.md

This project uses **CLAUDE.md** as the canonical AI guidance file.

For any agentic tool (Cursor, Copilot, Codex, Gemini, etc.):

1. Read `CLAUDE.md` for the project rules, folder map, and dev commands.
2. Drill into `docs/` for specific topics (`data-model.md`, `data-api.md`, `css-system.md`, `dashboard.md`).
3. Follow the same hard rules listed in `CLAUDE.md` (no npm, no Tailwind, no inline scripts, CSRF on POSTs, etc.).
